# Receipt Tracker

## Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/askw26-sketch/receipttracker.git
   cd receipttracker
   ```

2. **Install dependencies**:
   Depending on your package manager:
   - For npm:
   ```bash
   npm install
   ```
   - For yarn:
   ```bash
   yarn install
   ```

3. **Environment Configuration**:
   Make sure to configure your environment settings. Create a `.env` file based on the `.env.example` provided, and fill in the necessary keys.

## Usage

1. **Start the application**:
   ```bash
   npm start
   ```
   or
   ```bash
   yarn start
   ```

2. **Access the application**:
   Open your web browser and navigate to `http://localhost:3000`.

3. **Features**:
   - **Add Receipts**: Upload receipts and store them for future reference.
   - **View Receipts**: Browse through all uploaded receipts in a user-friendly interface.
   - **Filter/Sort Receipts**: Filter and sort receipts based on date, amount, or category.
   - **Export Data**: Export received data in various formats for personal use or accounting.

## Features

- User Authentication: Secure access to your receipts.
- Mobile Responsive: Access your receipts on any device.
- Multiple File Formats: Support for different receipt file formats (JPEG, PNG, PDF).
